## 📖 Overview

**SHOOTER: Cosmic Defender** is a premium HTML5 canvas-based space shooter game with stunning cyberpunk aesthetics. Take control of your advanced interceptor ship and defend the galaxy from waves of unstable meteors. Built with native rendering architectures, procedural parallax starfields, and particle physics collision engines.

## 🎮 Features

- **Immersive Gameplay**: Dodge and destroy waves of incoming meteors with auto-firing lasers
- **Cyberpunk Aesthetics**: Premium sci-fi typography with glowing UI elements and glass-morphism effects
- **High-Fidelity Audio**: Robust audio sync routines for immersive sound effects
- **Particle Physics**: Advanced collision detection and particle effects
- **Parallax Starfield**: Procedurally generated background for visual depth
- **Score Tracking**: Save your high scores and compete against yourself
- **Sound Toggle**: Control audio preferences from the main menu
- **Responsive Design**: Smooth gameplay across different screen sizes

## 🕹️ How to Play

### Controls
- **🖱️ Mouse Movement** - Move your interceptor ship left and right
- **🎯 Auto-Fire** - Lasers fire automatically as enemies approach
- **🎪 Objective** - Destroy all meteors and survive as long as possible!

### Game Flow
1. Launch from the main menu with **"LAUNCH MISSION"**
2. Guide your ship to destroy incoming meteors
3. Avoid collisions and maintain your defense grid
4. Rack up the highest score possible
5. Submit your score to the leaderboard

## 🚀 Getting Started

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/sainanuj/shooter.git
   cd shooter
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or visit the live demo: [https://sainanuj.github.io/shooter/](https://sainanuj.github.io/shooter/)

### Prerequisites
- A modern web browser with HTML5 Canvas support
- No dependencies or build tools required!
